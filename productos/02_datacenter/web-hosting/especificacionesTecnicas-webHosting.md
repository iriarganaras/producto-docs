# ESPECIFICACIONES TÉCNICAS Y ALCANCE DE SERVICIO - HOSTING

## 1. DESCRIPCIÓN GENERAL

El servicio HOSTING, en cualquiera de las variantes descriptas a continuación, ha sido especialmente diseñado para permitirle la publicación de un sitio Web en Internet utilizando la infraestructura de Datacenter que METROTEL posee para este fin. Al contratar los servicios de HOSTING de METROTEL accederá a importantes beneficios, entre los que podemos destacar:
+ Reemplazo de importantes inversiones en equipamiento y licencias de software por un accesible costo operativo mensual.
+ Reducción de riesgos operativos, asociados al manejo de la infraestructura.
+ Conectividad a Internet Nacional e Internacional multi-proveedor, permitiendo asegurar un alto nivel de disponibilidad de servicio.
+ Operación, soporte y mantenimiento del equipamiento a cargo de METROTEL.

## 3. SERVICIOS ADICIONALES

**Servicio DNS**

El servicio incluye un dominio dado de alta en los servidores DNS de METROTEL. DNS son las siglas de Domain Name System, un conjunto de protocolos y servicios que permiten a los usuarios de Internet utilizar nombres de dominio en vez de recordar direcciones IP. El servicio HOSTING requiere que exista, al menos, un nombre de dominio asociado a tu sitio Web. En el caso de requerir dominios adicionales, deberá contratarse un opcional de “servicio de DNS” por dominio.

**Registro y delegación de dominios**

El cliente deberá efectuar el trámite de Registración y/o Delegación de Dominio, ante la entidad administradora de dominios. Los dominios “.com.ar” son administrados exclusivamente por NIC Argentina (http://www.nic.ar). Los dominios “.com” son administrados por Network Solutions (http://www.netsol.com), Nic.com (http://www.nic.com), entre otras entidades. El trámite que se realiza ante el Ente Administrador de Dominios para dar de alta un nuevo dominio (Registración) o para realizar cambios sobre un dominio existente (Delegación), puede ser realizado vía Web.

**Cuentas de e-mail**

El servicio de HOSTING no contempla capacidad para cuentas de mail. Las mismas deberán ser contratadas en forma adicional al servicio.

**Soporte técnico**

Ubicado físicamente en la ciudad de Buenos Aires, el servicio de soporte técnico de METROTEL funciona las 24 horas del día, los 365 días del año. Para acceder al mismo, el cliente dispone de un número gratuito de contacto (0800-362-1040) mediante el cual podrá realizar la gestión de eventuales reclamos.

Desde la Mesa de Ayuda se realiza el diagnóstico para aislar la falla. Una vez identificada la misma, se procederá a realizar su inmediata resolución. El cliente será informado, ya sea por teléfono y/o mail, del estado y avance de la falla en particular, durante la existencia de la misma.

**Mantenimiento programado**

El mantenimiento programado consistirá en toda intervención realizada en la red de METROTEL, la cual será notificada previamente (con más de 48 horas de anticipación) al cliente y se realizará dentro del horario de mantenimiento establecido. Si el horario del corte previsto afecta la labor del cliente, éste puede solicitar la modificación del mismo. METROTEL se compromete a realizar los máximos esfuerzos para mover el corte al nuevo horario solicitado. Este tipo de interrupciones se puede realizar para reemplazar o agregar elementos a la red a fin de ampliar y mejorar permanentemente el servicio.

**Canales ODBC**

La variante de producto Basic Hosting sólo contiene drivers para canales ODBC a Bases de Datos Access. Para el caso de soporte de otras Bases de Datos que no sean Microsoft SQL o Linux My SQL se deberá consultar con el departamento técnico de METROTEL.

**Base de datos**

Para el tratamiento de Bases de Datos Microsoft SQL es necesario que el diseñador del sitio Web cuente con la herramienta Microsoft Enterprise Manager. La Base de Datos MS SQL no podrá ser accedida por un Master SQL; solamente podrá ser accedida localmente.

Para el tratamiento de Bases de Datos Linux My SQL el diseñador del sitio Web debe utilizar la herramienta PHP MyAdmin disponible en el Panel de Control que se entrega con el servicio. En caso de querer utilizar alguna herramienta de software Cliente My SQL, sólo lo podrá hacer para el producto Professional Hosting Linux de 2 GB o superior.

**Soporte PHP**

El soporte para el lenguaje de programación PHP está configurado teniendo en cuanta la versión 5 como default. Esto quiere decir que para utilizar la versión 3 ó 4, todos los archivos deberían contemplar la extensión .php3 ó .php4 respectivamente.

Ejemplo: suponiendo un archivo llamado index, deberá nombrase de la siguiente manera dependiendo de la versión de PHP que se desee utilizar.

1.  “index.php3” para la versión 3
2.  “index.php4” para la versión 4
3.  “index” para la versión 5

**Soporte ASP.NET**

El soporte del lenguaje de programación ASP.NET en el producto Professional Hosting Windows de 350 MB, sólo está disponible para el directorio raíz. En caso de necesitar soporte ASP.NET para todo el sitio Web deberás contratar la variante Professional Hosting Windows de 2 GB.

**Soporte de librerías y programas ejecutables**

Dado que se trata de una plataforma compartida el soporte de librerías y programas ejecutables deberá ser analizado y autorizado por el área de Ingeniería de Clientes de METROTEL. El único producto que soporta librerías o programas ejecutables es el Professional Hosting Windows de 2 GB.

## 4. REQUISITOS POR PARTE DEL CLIENTE

Queda bajo su responsabilidad la realización de los trámites correspondientes al dominio que se desee utilizar para el servicio CORREO ELECTRONICO (Registración, renovación y/o Delegación de Dominio). También, deberá indicar a METROTEL quién resolverá dicho dominio (por medio de los Servidores de DNS). En caso de que la resolución del dominio y/o el trámite de registración, renovación o delegación queden a cargo de METROTEL, por favor consulte a su Ejecutivo de Ventas para la contratación del servicio.

También queda bajo su responsabilidad procurarse de las herramientas y el conocimiento necesario para generar el sitio Web y su contenido. METROTEL, a través del Centro de Atención al Cliente, se concentra en el soporte de la Plataforma de Web Hosting, no brindando soporte para la generación y creación de contenido del sitio Web.

## 5. RESPONSABILIDAD POR CONTENIDOS

El cliente es responsable por los contenidos que genere y publique en este servicio, reservándose METROTEL el derecho de provocar la indisponibilidad del servicio en caso de que considere ofensivos los contenidos del sitio Web a la comunidad o parte de ella. El cliente, a su vez, es responsable por efectuar el Backup de su sitio, estando resguardado así ante el caso fortuito que los servidores de Hosting y sus respectivos servicios de Backup sufrieran daños irrecuperables. METROTEL no se hace responsable ante el caso fortuito de pérdida de datos en el servidor Web y su respectivo Backup. El cliente posee una clave de acceso para acceso vía FTP (File Transfer Protocol) a la gestión del sitio; dicha clave quedará en poder del cliente. En caso de que un tercero obtenga la clave y efectúe cambios, METROTEL no se hará responsable de las modificaciones o daños que este tercero hiciera al servicio.

## 6. POLÍTICAS DE SPAM

METROTEL no admite prácticas de SPAM por parte de sus clientes. Si METROTEL detectara tal situación, se procederá a dar de baja los servicios contratados. El cliente manifiesta y se compromete a no efectuar prácticas de SPAM a través de los servicios que contrata con METROTEL.
